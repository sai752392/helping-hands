About Us

A good complaint registration website allows user to register their complaints about anything from anywhere and it should be specific so that authorities can use the entered information effectively in future and can take actions to their complaints.

Overview

Our complaint registration website will allow to user to file their complaints in a simple way in different steps, asking about the type of problem, place from where the user is filing the complaint, from how many days he/she has been facing the problem and would ask the brief description about the situation. The users will also be provided with feedback page where he/she can give the feedback about their experience with the website
